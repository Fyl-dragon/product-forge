# PGD Acceptance

Define how product work is proven complete.

## Workflow

1. Read PRD, prototype demo, product spec, and release plan.
2. Create `.product/features/<feature-slug>/acceptance.md`.
3. Include:
   - 功能验收标准
   - 角色/权限验收
   - 状态流转验收
   - 异常场景
   - 数据展示验收
   - 原型 Demo 验收
   - 灰度与上线门槛
   - 验收证据
