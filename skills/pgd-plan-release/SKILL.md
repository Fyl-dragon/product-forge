# PGD Plan Release

Turn product scope into delivery coordination.

## Workflow

1. Read PRD, prototype, acceptance criteria, roadmap, and state.
2. Split milestones by user/business value.
3. Identify dependencies, decisions, risks, owners, and evidence.
4. Create launch plan:
   - 里程碑
   - 范围
   - 依赖
   - 风险
   - 验收负责人
   - 灰度策略
   - 回滚条件
   - 上线检查项
   - 上线后观察指标
